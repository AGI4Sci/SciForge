# Meta Skills

`packages/skills/meta_skills` contains skills for creating, debugging, routing, evaluating, and promoting other skills.

Use this directory for self-evolving skill workflows and capability maintenance tasks.

Rules:

- Treat generated task code as workspace-local until it is validated and promoted.
- Record evidence, tests, failure modes, and promotion criteria.
- Do not bypass package lifecycle boundaries when stabilizing a skill into observe/action/verifier/presentation/contract layers.
